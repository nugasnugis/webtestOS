Deployment & Installation Guide

Step 1

Download the Platform Image

Obtain the latest stable verified payload directly from our source library repository release index. Verify the SHA256 checksum profile checks before flashing.

Step 2

Flash Target Media Flashcards

Insert an empty USB Flashdrive (Minimum capacity 8 GB). Use image flash tools like BalenaEtcher or Rufus to build the boot partition architecture maps.

Step 3

Adjust Target BIOS Boot Rules

Restart your destination hardware system. Tap F12, F2, or Del immediately during startup to access your internal BIOS panel layout rules:

  • Set Secure Boot status parameter rules to: Disabled
  • Set boot order parameters to read your flashed USB Mass Storage drive first.
Step 4

Run Installer Utility Configurations

Boot directly into our live desktop window layer. Click on the system menu icon labeled "Install AxelOS" and follow the partition disk formatting wizard setup directions.
