AxelOS Core Features

Seamless Daily Driving

A beautiful, lightweight user interface pre-packed with essential apps for web browsing, office productivity, and media consumption.

Optimized for Gaming

Out-of-the-box support for Wine, Proton, and Steam. Custom kernel tweaks drastically reduce latency and maximize raw hardware FPS.

Advanced Pentesting

Equipped with a secure, sandboxed command-line shell environment and loaded with popular open-source network and system auditing suites.

About AxelOS

AxelOS started as a passion project to eliminate the need for computer enthusiasts to setup complicated dual-boots. By bundling standard daily desktop software, fine-tuned gaming runtime configurations, and top-tier security testing platforms natively into one distro, we provide a complete system built by power-users, for power-users.
