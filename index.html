Deployment & Installation Guide

Step 1

Download the Platform Image

Obtain the latest stable image and verify SHA256 checksums.

Step 2

Flash Target Media

Use tools like BalenaEtcher or Rufus to write the ISO to an 8GB+ USB drive.

Step 3

BIOS Configuration

Disable Secure Boot and set the USB drive as the primary boot device.

Step 4

Installation

Boot from USB and follow the on-screen installer instructions.
