Installation Guide

Step 1

Download Image

Download the latest verified image, then check the SHA256 sum.

Step 2

Flash USB

Use Rufus or BalenaEtcher to flash the image to a USB (min 8GB).

Step 3

BIOS Settings

Disable Secure Boot, then set BIOS to boot from USB.
